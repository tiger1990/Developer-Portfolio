- ## 🚀 Deploy Your Project with GitHub Pages

### 1️⃣ Push your project to GitHub

Make sure your project is already pushed to your repository.

**Example repo**

```
https://github.com/username/Developer-Portfolio
```

---

### 2️⃣ Open your repository

1. Go to your repository on GitHub  
2. Click **Settings** (top menu)

---

### 3️⃣ Go to GitHub Pages

1. In the left sidebar click **Pages**
2. Under **Build and deployment** configure:

- **Source:** Deploy from a branch  
- **Branch:** `main`  
- **Folder:** `/root`

Then click **Save**.

---

### 4️⃣ Wait for deployment

GitHub will build your site.

After **30–60 seconds**, your website will be available at:

```
https://username.github.io/Developer-Portfolio/
```

**Example**

```
https://tiger1990.github.io/Developer-Portfolio/
```

---

### 5️⃣ Verify your site

Open the link in your browser and your **portfolio website will be live 🌍**
